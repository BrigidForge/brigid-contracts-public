# Verification

Use this repo as the public source bundle for contract verification and source review.

## Recommended Public Verification Data

- deployed contract address
- network
- compiler version
- optimizer settings
- constructor arguments
- explorer verification link

## Suggested Process

1. Match the deployed address to the release or tag documented by Brigid Forge.
2. Confirm the contract source in this repo matches the verified source on the block explorer.
3. Confirm imported files and compiler settings are consistent with the public release metadata.

This file should stay concise and public-facing.
